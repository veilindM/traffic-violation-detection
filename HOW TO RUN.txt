Aplikasi Pendeteksi Plat Mobil di Jalan

Aplikasi pendeteksi plat mobil ini bekerja dengan cara mendeteksi mobil di jalan dan menangkap foto dari plat mobil kemudian palt yang sudah ter-record akan disimpan di database. Model yang digunakan untuk mendeteksi mobil dan plat mobil adalah library yolov9. Model ini kemudian dilakukan training kembali dengan dataset yang diambil dari kaggle.



venv : python -m venv venv (create), venv\Scripts\activate (activate)
frontend : streamlit run streamlit_app.py
backend : uvicorn app:app --reload
